Windows结构化异常处理SEH

2017年6月24日

Windows异常处理可以使用两种方式：筛选器异常处理和SEH异常处理，
今天介绍下SEH异常处理。

SEH(Structured Exception Handling)是Windows操作系统的一种系统机制，与程序语言无关。
在程序源代码中，通常使用__try、__except、__finally等关键字来具体实现。
注意windows异常处理关键字与C++中try catch等异常处理具有不同结构，不要混淆
